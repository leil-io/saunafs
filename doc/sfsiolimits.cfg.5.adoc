sfsiolimits.cfg(5)

== NAME

sfsiolimits.cfg - local I/O limiting configuration

== DESCRIPTION

The file *sfsiolimits.cfg* contains configuration of the local I/O limiter.

== SYNTAX,  OPTIONS

The same as in sfsglobaliolimits.cfg(5).
