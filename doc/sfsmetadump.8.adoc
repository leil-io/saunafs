sfsmetadump(8)

== NAME

sfsmetadump - a networking, distributed, highly available file system

== SYNOPSIS

[verse]
*sfsmetadump* 'metadata_file'...

== DESCRIPTION

*sfsmetadump*
dumps file system metadata into specified file.
