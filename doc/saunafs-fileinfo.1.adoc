saunafs-fileinfo(1)
====================

== NAME

saunafs-fileinfo - locate chunks

== SYNOPSIS

[verse]
*saunafs fileinfo* 'FILE'...

== DESCRIPTION

*fileinfo* prints location ('chunkserver' host and port) of each chunk copy belonging to
specified file(s). It can be used on any file, included deleted ('trash').
